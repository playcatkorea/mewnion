# 🇰🇷 Google Cloud Console 한글 메뉴 이름 정리

> Google Cloud Console이 한글로 나오는 경우 정확한 메뉴 이름

---

### 1️⃣ 최상단 메뉴

```

│    ↑             ↑                   │

│   메뉴          드롭다운               │

```

## 📋 정확한 한글 메뉴 이름

### 좌측 메뉴 (햄버거 메뉴 ☰ 클릭 후)

```
☰ 메뉴

│   ├─ OAuth 동의 화면
│   ├─ 라이브러리
│   └─ API 대시보드
├─ Compute Engine
└─ ...
```

---

## 🔑 OAuth 설정 관련 정확한 용어

### 메뉴 이름 (한글)
- **API 및 서비스** (메인 메뉴)
- **사용자 인증 정보** (Credentials)
- **OAuth 동의 화면** (OAuth consent screen)

### 버튼 이름 (한글)
- **+ 사용자 인증 정보 만들기** (+ CREATE CREDENTIALS)
- **만들기** (CREATE)
- **저장 후 계속** (SAVE AND CONTINUE)
- **확인** (OK)

### 옵션 이름 (한글)
- **OAuth 클라이언트 ID** (OAuth client ID)
- **웹 애플리케이션** (Web application)

### 필드 이름 (한글)
- **애플리케이션 유형** (Application type)
- **이름** (Name)
- **승인된 자바스크립트 원본** (Authorized JavaScript origins)
- **승인된 리디렉션 URI** (Authorized redirect URIs)
- **+ URI 추가** (+ ADD URI)

### 인증 정보 (한글)
- **클라이언트 ID** (Client ID)
- **클라이언트 보안 비밀번호** (Client Secret)

---

## 📝 OAuth 동의 화면 설정 용어

### User Type 선택 화면
- **Internal** → "내부 전용"
- **External** → "외부 사용자" ← 이거 선택!

### 앱 정보 입력 필드
- **앱 이름** (App name)
- **사용자 지원 이메일** (User support email)
- **개발자 연락처 정보** (Developer contact information)

---

## 🔍 단계별 정확한 메뉴 경로

### 1단계: OAuth 동의 화면 설정
```
☰ (햄버거 메뉴 클릭)
→ API 및 서비스 (마우스 올림)
  → OAuth 동의 화면 (클릭)
```

### 2단계: OAuth 클라이언트 ID 만들기
```
☰ (햄버거 메뉴 클릭)
→ API 및 서비스 (마우스 올림)
  → 사용자 인증 정보 (클릭)
    → + 사용자 인증 정보 만들기 (상단 버튼 클릭)
      → OAuth 클라이언트 ID (선택)
```

---

## ⚠️ 주의사항

### 메뉴 이름 정확히 찾기
Google Cloud Console은 계정 언어 설정에 따라 메뉴가 한글/영어로 나타납니다.

**확인 방법:**
1. 좌측 메뉴 열기 (☰)
2. "API 및 서비스" 또는 "APIs & Services" 찾기
3. 한글이면 위의 한글 메뉴 이름 사용
4. 영어면 괄호 안의 영어 이름 사용

## 🎯 빠른 체크리스트

설정할 때 이 순서대로 찾으세요:

### OAuth 동의 화면
- [ ] ☰ 메뉴 열기
- [ ] "API 및 서비스" 찾기
- [ ] "OAuth 동의 화면" 클릭
- [ ] "External" 선택
- [ ] 앱 이름: Mewnion
- [ ] 이메일 입력
- [ ] "저장 후 계속" 클릭

### OAuth 클라이언트 ID
- [ ] ☰ 메뉴 열기
- [ ] "API 및 서비스" 찾기
- [ ] "사용자 인증 정보" 클릭
- [ ] "+ 사용자 인증 정보 만들기" 클릭
- [ ] "OAuth 클라이언트 ID" 선택
- [ ] "웹 애플리케이션" 선택
- [ ] 이름 입력
- [ ] 승인된 자바스크립트 원본 추가
- [ ] 승인된 리디렉션 URI 추가 (Supabase callback URL)
- [ ] "만들기" 클릭
- [ ] 클라이언트 ID 복사
- [ ] 클라이언트 보안 비밀번호 복사
