# 🏷️ Google OAuth 로그인 시 표시되는 이름 변경하기

## 문제: "ylyypseqjlwsfakagbpn.supabase.co에 다시 로그인하는 중입니다"

Google 로그인 화면에서 이렇게 표시됩니다:
```
[앱 이름 또는 도메인]에 다시 로그인하는 중입니다
```

기본적으로 Supabase URL이 표시되는데, 이를 **서비스 이름(Mewnion)**으로 변경할 수 있습니다!

---

## ✅ 해결 방법

### 1️⃣ Google Cloud Console - OAuth 동의 화면 수정

#### 1-1. Google Cloud Console 접속
```
https://console.cloud.google.com
```

#### 1-2. 프로젝트 선택
화면 상단의 **[프로젝트 선택 ▼]** 클릭 → Mewnion 프로젝트 선택

#### 1-3. OAuth 동의 화면 메뉴로 이동

좌측 메뉴(☰) 클릭:
```
API 및 서비스
  └─ OAuth 동의 화면  ← 클릭!
```

**💡 영어 메뉴:** APIs & Services → OAuth consent screen

---

### 2️⃣ 앱 이름 변경

#### 2-1. 수정 버튼 클릭

OAuth 동의 화면 페이지에서:
```
┌────────────────────────────────┐
│ 앱 정보                         │
│ ─────────────────────────────  │
│ 앱 이름: Mewnion               │
│ 사용자 지원 이메일: ...        │
│                                 │
│              [앱 수정]  ← 클릭! │
└────────────────────────────────┘
```

**💡 영어 메뉴:** "EDIT APP" 버튼

---

#### 2-2. 앱 정보 섹션 확인

첫 번째 섹션 **"앱 정보"**에서:

```
┌─────────────────────────────────────┐
│ 앱 이름 *                            │
│ ┌─────────────────────────────────┐ │
│ │ Mewnion                         │ │ ← 여기!
│ └─────────────────────────────────┘ │
│                                      │
│ 앱 로고 (선택사항)                   │
│ [파일 선택]                          │
│                                      │
│ 사용자 지원 이메일 *                 │
│ [이메일 선택 ▼]                      │
└─────────────────────────────────────┘
```

**여기서 "앱 이름"을 확인/수정하세요:**
- ✅ **Mewnion** (권장)
- ✅ **묘연 Mewnion**
- ✅ **MEWNION - 고양이 미니홈피**

---

#### 2-3. (선택) 앱 로고 추가

사용자 경험을 더 좋게 하려면 로고를 추가할 수 있습니다:

**요구사항:**
- 크기: 120x120 픽셀 이상
- 형식: PNG 또는 JPG
- 정사각형
- 배경: 투명 또는 단색

**로고 추가하면:**
```
┌──────────────────────────────────┐
│     🐱 Mewnion                   │
│  logo  ↑                         │
│     에 로그인하는 중입니다        │
└──────────────────────────────────┘
```

---

#### 2-4. 저장

페이지 맨 아래로 스크롤:
```
[저장 후 계속]  ← 클릭
```

**💡 영어 메뉴:** "SAVE AND CONTINUE"

---

### 3️⃣ 앱 도메인 설정 (선택사항)

더 전문적으로 보이게 하려면 **앱 도메인**도 설정할 수 있습니다.

#### 3-1. OAuth 동의 화면 → 수정

같은 페이지에서 아래로 스크롤:

```
┌─────────────────────────────────────┐
│ 앱 도메인 (선택사항)                 │
│                                      │
```

**입력 후 저장**

## ⏱️ 반영 시간

- 변경 사항은 **즉시 반영**됩니다
- 간혹 **5-10분** 정도 걸릴 수 있습니다
- 캐시가 있다면 **시크릿 모드**에서 테스트하세요

---

## 🧪 테스트

### 1. 로그아웃
```
브라우저에서 현재 로그인된 계정 로그아웃
```

### 2. 시크릿 모드로 테스트
```
Ctrl + Shift + N (Chrome)
```

### 3. 로그인 시도
```
1. http://localhost:5173/login 접속
2. "Google로 로그인" 클릭
3. OAuth 화면 확인
```

### 4. 확인 사항
- ✅ "Mewnion에 로그인하는 중입니다" 표시
- ✅ 앱 로고가 보임 (설정했다면)
- ✅ Supabase URL이 안 보임

## 🔧 문제 해결

### "앱 수정" 버튼이 안 보여요
```
1. OAuth 동의 화면 페이지인지 확인
2. 올바른 프로젝트가 선택되었는지 확인
3. 페이지 새로고침
```

### 변경했는데 여전히 Supabase URL이 보여요
```
1. 브라우저 캐시 삭제
2. 시크릿 모드에서 테스트
3. 5-10분 대기 후 재시도
```

### "앱 이름이 이미 사용 중입니다" 에러
```
다른 이름 사용:
- Mewnion
- Mewnion App
- 묘연 Mewnion
- MEWNION 미니홈피
```
